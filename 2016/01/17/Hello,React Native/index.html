Hello,React Native

发表于 2016-01-17   |   分类于 React native   |  

前言

  • 上一节,我们讲了React native以及相关配套软件的安装.
  • 如果还有任何疑问,可以查看 React Native的安装, 或者在文章下面给我留言.
  • 请多关注我的个人网站

Action

1.创建项目

搭建好环境之后,我们就可以使用react-native命令来创建我们的第一个项目Hello了

  • 在终端输入命令行

    react-native init Hello

    此处有个注意点:命令是react-native,而不是react native.没错,我第一次的时候就犯过这个错
  • 第一次初始化的时候可能会比较慢,因为会去npm下载一些react native的相关的包
  • 如果一直卡在Installing react-native package from npm…这儿,可以将镜像改成淘宝的
  • 注意:两个命令逐一执行,缺一不可

npm config set registry https://registry.npm.taobao.org

npm config set disturl https://npm.taobao.org/dist

2.打开Hello项目

  • 在上一步创建项目成功之后,在最下方可以看到Open /Users/xxxx/Hello/ios/Hello/Hello.xcodeproj in Xcode
  • 找到上面的路径,双击Hello.xcodeproj打开
  • CMD+R运行即可.此时,终端会自动运行,不用管,也不要关闭.不出意外,你可以看到下面的画面

  • 如果很不幸,你看到的不是上面那样的.需要在你的终端输入ifconfig 查看自己本机的IP地址
  • 然后在AppDelegate.m文件中,将localhost改成你的IP地址即可.

3.修改index.ios.js文件

- 使用Sublime Text打开`index.ios.js`文件, 随意改一下fontSize或者margin之类的,然后选中你的模拟器,
CMD + R即可直接刷新.这就说明,我们修改JavaScript文件生效了.
